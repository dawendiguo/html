jQuery's event API is modeled after the standard event API put forth by the W3C. It's bind() methord simplefies this API, by requiring less code,and dropping a mostly unneeded argument.

jQuery doesn't stop thers, howerver, ou can also use individual event methods to bind events.

And if that weren't enough, jQuery also provides two methods that accommodate applying both a mouseover and mouseout event in the same method, and a method that makes toggling easier and more intuitive.

You can also programmatically trigger events, like automatically focusing the following <textarea> element on page load.

Even triggering an event is made easier, you can use individual, event methods whith no arguments to do that too.
